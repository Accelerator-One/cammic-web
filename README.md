### Usage
- Clone the repository
  ```bash
  git clone https://github.com/Accelerator-One/cammic-web.git
  ```

- Install npm dependencies
  ```bash
  npm install
  ```

- Running the build locally
  ```bash
  npm start
  ```

- Generating production build
  ```bash
  npm run build
  ```

- Generating component documentation
  ```bash
  npm run docs
  ```
